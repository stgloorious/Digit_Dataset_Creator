Detailed Description

creates a series of labeled pictures of digits for neural network training and testing.

This package creates pictures of digits that are in various fonts located in directory "/fonts", randomly rotated and positioned. Some random noise gets added to every picture. The output is in .csv-Format, like the digits in the MNIST database.
